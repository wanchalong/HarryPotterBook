
## Step to use Web HarryPotterBook Shopping
### 1. Clone
```
$ git clone https://github.com/wanchalong/HarryPotterBook.git
```
### 2. Install
```
$ npm install
```
### 3. Run
```
$ npm start
```
### 4. Test
```
$ npm test
```
